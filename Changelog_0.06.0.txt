Fix 0.06.0.01 - 0.06.0.05

0.06.0.01
 - fixed a bug that occurred when it was impossible to determine the size of the game window (исправлена ошибка, возникавшая при невозможности определения размеров окна с игрой);
 - fixed a bug when entering Ann's room when Eric is there (исправлена ошибка при входе в комнату Анны, когда там Эрик);
 - fixed a bug when massaging Alice in the yard (исправлена ошибка при массаже Алисы во дворе);
 - fixed an issue with an empty dialog box (исправлена ошибка с пустым диалоговым окном);
 - fixed a bug with missing images (исправлена ошибка с отсутствующими изображениями);
 - fixed a bug that occurred at midnight in the first weeks of the game (исправлена ошибка, возникавшая в полночь в первые недели игры);
 - fixed an issue where Max could give Alice a lace bodysuit multiple times. (исправлена ошибка, из-за которой Макс мог подарить Алисе кружевное боди несколько раз).

0.06.0.02
 - if Eric gave Alice a lace bodysuit, then it is removed from Max's things (если Эрик подарил Алисе кружевное боди, то оно убирается из вещей Макса).

0.06.0.03
 - fixed a bug that caused Alice and Kira to return from the club several times during the night (исправлена ошибка, из-за которой возвращение Алисы и Киры из клуба происходило несколько раз за ночь).

0.06.0.04
 - fixed a bug where Ann slept in her underwear when Eric left the room (исправлена ошибка, когда Анна спала в нижнем белье при уходе Эрика из комнаты);
 - fixed a bug that occurred when selecting a massage area on Alice in the yard (исправлена ошибка, возникавшая при выборе зоны массажа на Алисе во дворе);
 - fixed a bug with missing images on the laptop screen when watching horror movies (исправлена ошибка с отсутствием изображений на экране ноутбука при просмотре ужастиков);
 - fixed a bug with mixed up images during lessons for Lisa in Ann's room (исправлена ошибка с перепутанными изображениями при уроках для Лизы в комнате Анны);
 - fixed a bug that caused some conversations with Eric to repeat (исправлена ошибка, из-за которой некоторые диалоги с Эриком повторялись);
 - fixed an issue where Eric accused Max of giving a lace bodysuit to Alice, but he gave it to her himself (исправлена ошибка, когда Эрик обвинял Макса в опережении с кружевным боди, но при этом сам его дарил);
 - in Android versions, the cursor is removed (в Андройд-версиях убран курсор).
